# Security

Report security issues privately to `security@ledgerbill.app`.

For non-sensitive bugs and feature requests, use: https://github.com/no1rstack/ledgerbill-sdk/issues

Do not include production API keys, tenant identifiers, webhook secrets, or customer data in public issues.

The extension stores LedgerBill API keys in VS Code Secret Storage and uses only customer-facing LedgerBill APIs. It does not expose Platform Admin, diagnostics, key rotation, route governance, or LedgerBill team-only endpoints.
